Trending Now Report Reference

In this topic you will learn about the Trending Now report that is provided as part of the Analytics module.
Note: This feature is in Limited Availability. If you are interested in enabling it for your account, contact your Account Manager.

To view the Trending Now report, open the Analytics module and select Trending Now from the left side navigation. The report will display a graph showing the number of concurrent viewers for the past hour. The graph will update every minute or every time you reload the page.

Note: A unique user is determined by browser user agent combined with IP address. Multiple tabs of the same player running in the same browser will only be 1 user.

If your account is enabled for Brightcove Live, you will have the option to view the number of concurrent viewers for all videos, VOD assets or live assets.

Sections below the graph will display the Top Videos, Top Countries and Top Cities for the last minute and will update every 45 seconds. The percentages in the TOP Videos, Countries and Cities lists are calculated based on the top 10 items.
